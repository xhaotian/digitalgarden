α2受体激动剂
# 作用机制
主要是==抑制产生==
**一进一出**（减少生成，增加葡萄膜巩膜突途径流出）

# 不良反应
局部刺激、过敏、瞳孔散大、口干、眼干、低血压、嗜睡

# **禁忌症**
[[Boxes/单胺氧化酶抑制剂\|MAOI]]使用者
**酒石酸溴莫尼定不用于<2岁儿童**----儿童呼吸暂停

# 评价
对氨基可乐定：可用于**短期使用**和**预防激光后的眼压高峰**
