# 最大肌力训练
强度：≥85%，≤6RM，组间3-5min（11组？）
供能：磷酸原为主，也有无氧功能

# 肌肥大训练
强度：67%-85%，6-12RM，组间90s（4组？）
供能：糖酵解（无氧）为主


# 肌耐力
强度<67%，≥12RM
供能：有氧为主，也有无氧
