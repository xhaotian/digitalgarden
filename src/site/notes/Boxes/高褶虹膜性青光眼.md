# 解剖结构
虹膜根部插入插在睫状体上，虹膜周边部角状高褶、向前再转向瞳孔区
特征：**窄**房角，但虹膜**平坦**、前房**不浅**

# 流病特点
女多见，常有闭青家族史，较瞳孔阻滞型闭青者年轻，多30-50
# 发病形式
房角**可自发关闭**，或**瞳孔扩大**后关闭，尤其虹膜**周切术后瞳孔扩大**仍会引起房角关闭。

# 依据虹膜褶高度和范围分类：
完全性：褶较高且一周均有，眼压急性升高，类急闭青发作
不完全性：褶较低且不完整，慢性病程、类似慢闭表现。

# 诊断
明确：虹膜**周切后暗室实验**阳性
**光线**与方角：房角暗光下关闭，亮光下开放

# 治疗
**缩瞳**/周边**虹膜成形**术-->拉平高褶
已有白内障指征：做单纯白内障手术——**白**
已发生广泛周边粘连：滤过性手术——**大青白**
